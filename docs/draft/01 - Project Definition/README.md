# Project Definition

## Stakeholders and Goals

### Goal G1

The system to be implemented is a framework and must enable an application developer to access different blockchain
ledgers via a single interface.

| Stakeholder           | Interest(s)                                                                                                                                                                                           |
|-----------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Application Developer | a) The framework must be easy to use. <br/>b) The framework must have reusable interfaces and components<br/>c) The framework must be configurable and adaptoble to multiple application requirements |

### Goal G2

The system to be implemented must enable price inquiries and trading for various liquidity providers (exchanges).
